Hyperbolic
Functions

双曲函数完整复习 — 从定义到反函数，从恒等式到解方程，覆盖所有考点与易错点。

$$\sinh x = \frac{e^x - e^{-x}}{2}$$ $$\cosh x = \frac{e^x + e^{-x}}{2}$$ $$\tanh x = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
01 · Definitions
定义与指数形式

双曲函数来源于单位双曲线 \(x^2 - y^2 = 1\)，与三角函数来源于单位圆 \(x^2 + y^2 = 1\) 相对应。所有双曲函数都用 \(e^x\) 来定义。

P
Primary Functions 主要函数
sinh x $$\sinh x = \frac{e^x - e^{-x}}{2}$$
cosh x $$\cosh x = \frac{e^x + e^{-x}}{2}$$
tanh x $$\tanh x = \frac{\sinh x}{\cosh x} = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
R
Reciprocal Functions 倒数函数
cosech x $$\operatorname{cosech} x = \frac{1}{\sinh x} = \frac{2}{e^x - e^{-x}}$$
sech x $$\operatorname{sech} x = \frac{1}{\cosh x} = \frac{2}{e^x + e^{-x}}$$
coth x $$\operatorname{coth} x = \frac{\cosh x}{\sinh x} = \frac{e^x + e^{-x}}{e^x - e^{-x}}$$
关键关系式

$$e^x = \cosh x + \sinh x \qquad e^{-x} = \cosh x - \sinh x$$

把 sinh 和 cosh 的定义直接相加/相减即可得到。这个关系在化简和证明中经常用到。

Example 1.1 求 \(\sinh(\ln 2)\) 和 \(\tanh(\ln 4)\) 的值 +

(i) \(\sinh(\ln 2)\)：

代入定义

$$\sinh(\ln 2) = \frac{e^{\ln 2} - e^{-\ln 2}}{2} = \frac{2 - \frac{1}{2}}{2} = \frac{\frac{3}{2}}{2} = \frac{3}{4}$$

(ii) \(\tanh(\ln 4)\)：

代入定义

$$\tanh(\ln 4) = \frac{e^{\ln 4} - e^{-\ln 4}}{e^{\ln 4} + e^{-\ln 4}} = \frac{4 - \frac{1}{4}}{4 + \frac{1}{4}} = \frac{\frac{15}{4}}{\frac{17}{4}} = \frac{15}{17}$$

技巧

遇到 \(e^{\ln a}\) 直接等于 \(a\)，而 \(e^{-\ln a} = \frac{1}{a}\)。这是此类题的核心简化。

Example 1.5 若 \(\sinh x = \frac{3}{4}\)，求 sech x, tanh x, cosh 2x, tanh 2x +
Step 1 · 求 cosh x

由 \(\cosh^2 x - \sinh^2 x = 1\)：

$$\cosh^2 x = 1 + \frac{9}{16} = \frac{25}{16} \implies \cosh x = \frac{5}{4}$$

注意：\(\cosh x \geq 1\)，所以 \(\cosh x = -\frac{5}{4}\) 被 reject。

Step 2 · 求 sech x 和 tanh x

$$\operatorname{sech} x = \frac{1}{\cosh x} = \frac{4}{5}$$

$$\tanh x = \frac{\sinh x}{\cosh x} = \frac{3/4}{5/4} = \frac{3}{5}$$

Step 3 · 双角公式

$$\cosh 2x = \cosh^2 x + \sinh^2 x = \frac{25}{16} + \frac{9}{16} = \frac{34}{16} = \frac{17}{8}$$

$$\tanh 2x = \frac{2\tanh x}{1 + \tanh^2 x} = \frac{2 \times \frac{3}{5}}{1 + \frac{9}{25}} = \frac{\frac{6}{5}}{\frac{34}{25}} = \frac{6}{5} \times \frac{25}{34} = \frac{15}{17}$$

Example 1.6 若 \(\tanh x = \frac{5}{13}\)，求 cosech x, cosh x, sinh 2x, tanh 2x +
Step 1 · 求 cosech x

由 \(\operatorname{coth}^2 x - 1 = \operatorname{cosech}^2 x\)，且 \(\operatorname{coth} x = \frac{13}{5}\)：

$$\operatorname{cosech}^2 x = \frac{169}{25} - 1 = \frac{144}{25} \implies \operatorname{cosech} x = \frac{12}{5}$$

Step 2 · 求 sinh x 和 cosh x

$$\sinh x = \frac{5}{12}, \quad \cosh x = \frac{13}{12}$$

\(\cosh x = -\frac{13}{12}\) reject，因为 \(\cosh x \geq 1\)。

Step 3 · 双角公式

$$\sinh 2x = 2\sinh x \cosh x = 2 \times \frac{5}{12} \times \frac{13}{12} = \frac{65}{72}$$

$$\tanh 2x = \frac{2\tanh x}{1 + \tanh^2 x} = \frac{2 \times \frac{5}{13}}{1 + \frac{25}{169}} = \frac{\frac{10}{13}}{\frac{194}{169}} = \frac{65}{97}$$

cosh x 不是一对一函数！

cosh x 是偶函数，图像关于 y 轴对称。要让它有反函数，必须限制定义域为 \(x \geq 0\)（或 \(x \leq 0\)）。这就是为什么 \(\cosh^{-1} x\) 只取正值。

如何记住图像

sinh x 是 \(\frac{1}{2}e^x\) 和 \(-\frac{1}{2}e^{-x}\) 的"平均"，像被拉伸的 S 形。

cosh x 是 \(\frac{1}{2}e^x\) 和 \(\frac{1}{2}e^{-x}\) 的"平均"，形成悬链线（catenary）— 悬挂链条的形状。

tanh x 被两条水平渐近线 \(y = \pm 1\) 夹住，像一个被压扁的 S 形。

Example 1.3 画 \(y = |3\cosh(x+1) - 4|\) 的图像 +

设 \(f(x) = \cosh x\)，按变换顺序逐步画图：

Step 1 · 水平平移

\(y = f(x+1) = \cosh(x+1)\)：图像向左平移 1 个单位

Step 2 · 垂直拉伸

\(y = 3f(x+1) = 3\cosh(x+1)\)：图像在 y 方向拉伸 3 倍，最小值变为 3

Step 3 · 垂直平移

\(y = 3f(x+1) - 4 = 3\cosh(x+1) - 4\)：图像向下平移 4 个单位，最小值变为 \(3 - 4 = -1\)

Step 4 · 取绝对值

\(y = |3\cosh(x+1) - 4|\)：把 y 轴下方的部分翻折到上方。最小值处形成"V"形尖角。

画图要标注的关键信息

最低点坐标、与坐标轴的交点、渐近线（如果有）、对称轴。考试要求标出这些。

03 · Identities
双曲恒等式
≡
三大核心恒等式
$$\cosh^2 x - \sinh^2 x \equiv 1$$
$$1 - \tanh^2 x \equiv \operatorname{sech}^2 x$$
$$\operatorname{coth}^2 x - 1 \equiv \operatorname{cosech}^2 x$$

第一个除以 \(\cosh^2 x\) 得第二个，除以 \(\sinh^2 x\) 得第三个。

2x
双角公式
$$\sinh 2x = 2\sinh x \cosh x$$
$$\cosh 2x = \cosh^2 x + \sinh^2 x = 2\cosh^2 x - 1 = 1 + 2\sinh^2 x$$
$$\tanh 2x = \frac{2\tanh x}{1 + \tanh^2 x}$$
+
加法公式 (Addition Formulae)
$$\sinh(A + B) \equiv \sinh A \cosh B + \cosh A \sinh B$$
$$\cosh(A + B) \equiv \cosh A \cosh B + \sinh A \sinh B$$
$$\tanh(A + B) \equiv \frac{\tanh A + \tanh B}{1 + \tanh A \tanh B}$$
⚡
Osborne's Rule

从三角恒等式快速转换为双曲恒等式的规则：

sin → sinh，cos → cosh，tan → tanh

⚠️ 凡是涉及两个 sinh 相乘（即 sinh 的"积"）的项，要变号。

例：\(\cos^2 x + \sin^2 x = 1\) → \(\cosh^2 x - \sinh^2 x = 1\)

因为 \(\sin^2 x\) 是两个 sin 的积，变号变成 \(-\sinh^2 x\)

例：\(\tan(A+B) = \frac{\tan A + \tan B}{1 - \tan A \tan B}\) → \(\tanh(A+B) = \frac{\tanh A + \tanh B}{1 + \tanh A \tanh B}\)

因为 \(\tan A \tan B = \frac{\sinh A}{\cosh A} \cdot \frac{\sinh B}{\cosh B}\) 含两个 sinh 的积，所以分母的负号变正号

Osborne's Rule 不是证明！

考试中如果要求 "prove"（证明），你必须从指数定义出发推导，不能引用 Osborne's Rule。这个规则只能在 "state"（陈述）或 "write down"（写出）类题目中使用。

Example 1.4 证明 \(\cosh^2 x - \sinh^2 x \equiv 1\) PROOF 高频 +
LHS

$$\cosh^2 x - \sinh^2 x = \frac{(e^x + e^{-x})^2}{4} - \frac{(e^x - e^{-x})^2}{4}$$

$$= \frac{(e^x + e^{-x})^2 - (e^x - e^{-x})^2}{4}$$

利用 \(a^2 - b^2 = (a+b)(a-b)\)：

$$= \frac{(2e^x)(2e^{-x})}{4} = \frac{4e^0}{4} = \frac{4}{4} = 1 = \text{RHS}$$

也可以直接展开

展开两个平方后相减，\(e^{2x}\) 和 \(e^{-2x}\) 项会全部消掉，只剩常数 \(\frac{4}{4} = 1\)。两种方法都可以。

Example 1.4(ii) 证明 \(\tanh 2x = \frac{2\tanh x}{1 + \tanh^2 x}\) PROOF +
从 RHS 出发

$$\text{RHS} = \frac{2 \cdot \frac{e^x + e^{-x}}{e^x - e^{-x}}}{1 + \frac{(e^x + e^{-x})^2}{(e^x - e^{-x})^2}} = \frac{\frac{2(e^x + e^{-x})}{e^x - e^{-x}}}{\frac{(e^x - e^{-x})^2 + (e^x + e^{-x})^2}{(e^x - e^{-x})^2}}$$

$$= \frac{2(e^x + e^{-x})}{e^x - e^{-x}} \cdot \frac{(e^x - e^{-x})^2}{(e^x - e^{-x})^2 + (e^x + e^{-x})^2}$$

$$= \frac{2(e^x + e^{-x})(e^x - e^{-x})}{2(e^{2x} + e^{-2x})}$$

$$= \frac{2(e^{2x} - e^{-2x})}{2(e^{2x} + e^{-2x})} = \frac{e^{2x} - e^{-2x}}{e^{2x} + e^{-2x}} = \tanh 2x = \text{LHS}$$

04 · Solving Equations
解双曲方程

解双曲方程有两种主要方法，考试可能指定使用哪一种。

A
方法一：指数法（化成 eˣ 的二次方程）

把所有双曲函数用 \(e^x\) 的定义替换，乘以 \(2e^x\)（或 \(2\)）消去分母，得到关于 \(e^x\) 的二次方程，然后解之。

注意：\(e^x > 0\)，所以 \(e^x\) 的负数解必须 reject。

B
方法二：恒等式法（化成双曲函数的二次方程）

用恒等式统一成同一个双曲函数，然后解关于该函数的二次方程。最后用反双曲函数或对数形式求 \(x\)。

注意 range 限制：\(\cosh x \geq 1\)，\(-1 < \tanh x < 1\)。超出范围的解必须 reject。

Example 1.2(i) 解 \(19\sinh x + 16\cosh x = 8\) 高频题型 +
代入指数定义

$$19 \cdot \frac{e^x - e^{-x}}{2} + 16 \cdot \frac{e^x + e^{-x}}{2} = 8$$

乘以 2 并整理

$$19e^x - 19e^{-x} + 16e^x + 16e^{-x} = 16$$

$$35e^x - 3e^{-x} = 16$$

乘以 eˣ 得二次方程

$$35e^{2x} - 16e^x - 3 = 0$$

$$(5e^x - 3)(7e^x + 1) = 0$$

求解

\(e^x = \frac{3}{5}\) → \(x = \ln\frac{3}{5}\)

\(e^x = -\frac{1}{7}\) → reject（\(e^x > 0\)）

Example 19.2 解 \(4\cosh x - 3\sinh x = 4\)（课本） +
代入指数定义，乘以 2

$$4(e^x + e^{-x}) - 3(e^x - e^{-x}) = 8$$

$$4e^x + 4e^{-x} - 3e^x + 3e^{-x} = 8$$

$$e^x + 7e^{-x} = 8$$

乘以 eˣ

$$e^{2x} - 8e^x + 7 = 0$$

$$(e^x - 7)(e^x - 1) = 0$$

求解

\(x = \ln 7\) 或 \(x = \ln 1 = 0\)

Example 19.3 解 \(3\tanh^2 x - 4\tanh x = 4\)（恒等式法） +
令 T = tanh x

$$3T^2 - 4T - 4 = 0$$

$$3T^2 - 6T + 2T - 4 = 0$$

$$(T-2)(3T+2) = 0$$

判断解的有效性

\(T = 2\) → reject！因为 \(-1 < \tanh x < 1\)，不可能等于 2。

\(T = -\frac{2}{3}\) → 有效，\(x = \tanh^{-1}\left(-\frac{2}{3}\right) = \frac{1}{2}\ln\frac{1/3}{5/3} = \frac{1}{2}\ln\frac{1}{5}\)

Example 1.8(i) 解 \(\sinh^2 x - 5\sinh x + 4 = 0\) +
因式分解

$$(\sinh x - 4)(\sinh x - 1) = 0$$

\(\sinh x = 4\) 或 \(\sinh x = 1\)

用反函数求解

\(x = \sinh^{-1} 4 = \ln(4 + \sqrt{17})\)

\(x = \sinh^{-1} 1 = \ln(1 + \sqrt{2})\)

注意

sinh x 的 range 是全体实数，所以 sinh x = 4 和 sinh x = 1 都是有效的，不需要 reject。

Example 1.8(ii) 解 \(2\cosh 2x = 9\cosh x - 7\) +
用双角公式

\(\cosh 2x = 2\cosh^2 x - 1\)，代入：

$$2(2\cosh^2 x - 1) = 9\cosh x - 7$$

$$4\cosh^2 x - 9\cosh x + 5 = 0$$

因式分解

$$(4\cosh x - 5)(cosh x - 1) = 0$$

\(\cosh x = \frac{5}{4}\) 或 \(\cosh x = 1\)

求解

\(x = \cosh^{-1}\frac{5}{4} = \ln\left(\frac{5}{4} + \sqrt{\frac{25}{16} - 1}\right) = \ln 2\)

\(x = \cosh^{-1} 1 = 0\)

Example 1.8(iii) 解 \(4\cosh x - \sinh x = 8\)（要求精确对数形式） +
代入指数定义

$$4 \cdot \frac{e^x + e^{-x}}{2} - \frac{e^x - e^{-x}}{2} = 8$$

$$\frac{4e^x + 4e^{-x} - e^x + e^{-x}}{2} = 8$$

$$3e^x + 5e^{-x} = 16$$

乘以 eˣ

$$3e^{2x} - 16e^x + 5 = 0$$

$$(3e^x - 1)(e^x - 5) = 0$$

求解

\(e^x = \frac{1}{3}\) → \(x = -\ln 3 = \ln\frac{1}{3}\)

\(e^x = 5\) → \(x = \ln 5\)

Example 1.7 / 19.8 推导 \(\sinh^{-1} x = \ln(x + \sqrt{x^2 + 1})\) PROOF 高频 +
设 y = sinh⁻¹ x

则 \(\sinh y = x\)，即 \(\frac{e^y - e^{-y}}{2} = x\)

Example 1.9 / 19.10 证明 \(\sinh^{-1} A + \sinh^{-1} B = \sinh^{-1}(A\sqrt{1+B^2} + B\sqrt{1+A^2})\) 少见 +

07 · Common Traps
易错点与陷阱
1
忘记 reject 无效解

这是最常见的失分点。三个必须检查的约束：

• \(e^x > 0\) — 指数法中 \(e^x\) 不能为负数

• \(\cosh x \geq 1\) — 恒等式法中 \(\cosh x < 1\) 的解无效

• \(-1 < \tanh x < 1\) — 恒等式法中 \(|\tanh x| \geq 1\) 的解无效

2
Osborne's Rule 中的变号搞错

只有两个 sinh 的积才变号。单独的 sinh 不变号，cosh 不变号。

例如：\(\sin^2 x\) 是 \(\sin x \cdot \sin x\)，两个 sin 相乘，变号 → \(-\sinh^2 x\)

但 \(\sin 2x = 2\sin x \cos x\) 只有一个 sin，不变号 → \(\sinh 2x = 2\sinh x \cosh x\)

3
对数形式推导中取错根

推导 \(\sinh^{-1} x\) 时，求根公式给出 \(e^y = x \pm \sqrt{x^2 + 1}\)。

必须论证为什么取正号：因为 \(\sqrt{x^2+1} > |x|\)，所以 \(x - \sqrt{x^2+1} < 0\)，而 \(e^y > 0\)。

考试中不写这个论证过程会扣分！

4
混淆 cosh 2x 的三种形式

\(\cosh 2x = \cosh^2 x + \sinh^2 x = 2\cosh^2 x - 1 = 1 + 2\sinh^2 x\)

选择哪一种取决于方程中已有的函数类型。如果方程含 cosh x，用 \(2\cosh^2 x - 1\)；含 sinh x，用 \(1 + 2\sinh^2 x\)。

5
Proof 题中用 Osborne's Rule

如果题目说 "prove" 或 "show"，你必须从指数定义出发推导。直接写 "By Osborne's Rule" 拿不到分。

Osborne's Rule 只能用于 "state" 或 "write down" 类题目。

6
求精确值时写成小数

题目要求 "exact form" 时，答案必须是对数形式如 \(\ln(4 + \sqrt{17})\)，不能写 2.09。

如果题目要求 "3 significant figures"，则需要用计算器算出近似值。

08 · Deeper Insight
二次洞察与举一反三
为什么叫"双曲"函数？

三角函数参数化单位圆 \(x^2 + y^2 = 1\)：令 \(x = \cos t, y = \sin t\)。

双曲函数参数化单位双曲线 \(x^2 - y^2 = 1\)：令 \(x = \cosh t, y = \sinh t\)。

验证：\(\cosh^2 t - \sinh^2 t = 1\) ✓ — 这就是核心恒等式的几何意义。

cosh x 的物理意义：悬链线 (Catenary)

一条均匀的链条在重力下自然悬挂的形状恰好是 \(y = a\cosh\frac{x}{a}\)。圣路易斯拱门（Gateway Arch）的形状就是倒置的悬链线。这不是抛物线！虽然看起来很像，但数学性质完全不同。

sech²x 与孤立子波 (Soliton)

\(y = \operatorname{sech}^2 x\) 的图形被用来建模海啸和光纤中的孤立子波 — 一种传播时不改变形状、不损失能量的特殊波。它的速度与振幅成正比，这就是海啸具有巨大破坏力的原因之一。

举一反三：从一个证明方法到多个

证明 \(\cosh^2 x - \sinh^2 x \equiv 1\) 有至少三种方法：

方法 1：直接展开平方，然后相减（最直接）

方法 2：用 \(a^2 - b^2 = (a+b)(a-b)\)（更优雅）

方法 3：用 \(e^x = \cosh x + \sinh x\) 和 \(e^{-x} = \cosh x - \sinh x\)，两者相乘得 \(e^0 = \cosh^2 x - \sinh^2 x = 1\)（最巧妙）

考试中选择你最熟练的方法，但了解多种方法能加深理解。

为什么 cosh⁻¹ x 的对数形式只取正根？

推导 \(\cosh^{-1} x\) 时，求根公式给出 \(e^y = x \pm \sqrt{x^2 - 1}\)。注意此时 \(x \geq 1\)。

\(x - \sqrt{x^2 - 1} = \frac{1}{x + \sqrt{x^2-1}}\)（有理化），所以它确实是正数。

但我们规定 \(\cosh^{-1} x \geq 0\)（因为限制了 \(y \geq 0\)），而 \(\ln(x - \sqrt{x^2-1}) < 0\)，所以取 \(e^y = x + \sqrt{x^2-1}\)。

这与 sinh⁻¹ x 的情况不同 — 那里负根确实小于零所以被 reject，而这里负根虽然是正数但给出的 y 值为负。
